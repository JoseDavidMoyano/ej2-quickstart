This demo showcases how to customize the starting week of the month and also customize the number of weeks to be rendered in month view.

In this demo, the starting week of the month view can be customized by using the displayDate property and also the number of weeks to be rendered in a month view can be customized by using the numberOfWeeks property. These properties have been configured under the views property view options. displayDate and numberOfWeeks properties can be applicable only for month view.
